<p>
  <img alt="Version" src="https://img.shields.io/badge/version-0.0.1-blue.svg?cacheSeconds=2592000" />
</p>

> This project represents the frontend of web based application is a prototype for a crypto asset tracker, meaning that it allows users to search for coins and tokens they may have and arrange in list. Each list item represents a coin and contains its holdings which can be added to a storage type (wallet, exchange or staking). The items are summarised to calculate the total value of the users holdings, in either GBP, USD or EUR. The app is also available in several languages, German and English and the moment.

### 🏠 [Homepage](https://crypto-asset-tracker-client-dev.up.railway.app/)

## Install

```sh
npm i
```

## Usage

```sh
npm run dev
```

## Run tests

```sh
npm run test // unit tests
npm run cy:test // e2e tests
```
